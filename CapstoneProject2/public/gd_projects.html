Simple Stock Checker (SSC)

This Python application processes current stock financial data and analyst data and grades the stock. The user can choose a text file with comma separated ticker symbols and have the tool process a grade for each. For each ticker symbol a grade-sheet xlsx is stored and all pertinent information is stored in a local SQL database for review.

I intend on upgrading this application in the future to allow for full user control over the grading mechanism so it can be tweaked to their personal metrics.

This project gave me more insight into OOP concepts and experience using requests module, mySQL, Pandas, Shelve and more.
